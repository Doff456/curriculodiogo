Projeto de Coleta de Dados Bancários no LinkedIn

Scraping automatizado com exportação de dados e interface gráfica integrada

Contexto

Muitas empresas precisam encontrar e analisar informações sobre profissionais do setor bancário, seja para contratar, estudar o mercado ou buscar contatos estratégicos. A principal fonte desses dados costuma ser o LinkedIn, uma rede social voltada para o mundo do trabalho.

No entanto, o LinkedIn impõe diversas limitações: é necessário estar logado, há restrições de acesso por quantidade de perfis vistos, e o uso de sua API oficial é limitado, burocrático e, muitas vezes, pago. Isso dificulta bastante qualquer tentativa de análise em escala.

Este projeto foi criado justamente para contornar esses obstáculos de forma segura e automatizada. A ferramenta simula o comportamento de um usuário comum, acessando perfis públicos de forma discreta e extraindo informações úteis como nome completo, CPF (quando disponível em fontes abertas), número de telefone, cargo, empresas em que trabalhou e por quanto tempo.

Um dos maiores diferenciais do projeto é sua independência: ele não depende de APIs do LinkedIn, de autenticação ou de ferramentas externas para funcionar. Toda a lógica de coleta, organização e exportação dos dados é feita internamente, usando apenas recursos controlados diretamente pelo sistema.

Além disso, o projeto já conta com uma interface gráfica, exportação de resultados e um painel visual para facilitar o uso, mesmo por quem não tem conhecimento técnico.

Abordagem Técnica

  • Coleta de links de perfis via Google Search API com filtros por setor, cargo e região
  • Extração automatizada com Puppeteer simulando navegação humana
  • Burlas implementadas: rotação de user-agents, proxies, delays aleatórios, controle de sessão
  • Coleta de dados públicos e complementares obtidos de fontes abertas

Tipos de Dados Coletados

  • Nome completo
  • CPF (quando disponível em fontes públicas correlatas)
  • Número de telefone
  • Cargo atual e anteriores
  • Empresas onde trabalhou e período de atuação
  • Localização e breve descrição do perfil

Funcionalidades Implementadas

Scraping com Puppeteer
Exportação em CSV, JSON e Excel
Interface gráfica completa
Dashboard com filtros e busca
Agendamento automático
Controle de sessão e reintentos

Diferenciais Técnicos

  • Independência de APIs privadas
  • Alta taxa de sucesso mesmo em perfis com restrições regionais
  • Foco específico no nicho bancário
  • Coleta silenciosa e segura de dados públicos e complementares

Próximos Desenvolvimentos

  • Integração com sistemas externos (ATS, CRM, BI)
  • Notificações e relatórios automatizados
  • Detecção de padrões e gaps em perfis
Projeto com fins estratégicos e educacionais. Apenas dados públicos e fontes abertas são utilizados.
